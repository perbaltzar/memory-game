# Meme-ory
A Memory game made by Per Baltzar
[<img src="https://media.giphy.com/media/ppxpN9gL1OEpO/giphy.gif" width="100%">](https://media.giphy.com/media/ppxpN9gL1OEpO/giphy.gif)

Made in Javascript, CSS and HTML

## How to play
1. [Clone](https://github.com/perbaltzar/memory-game) this in GitHub Desktop,
`$ git clone https://github.com/perbaltzar/memory-game.git` in bash or by visiting
2. Open index.html
3. You probably know the rules, but turn over the cards and match them.
4. Play!
